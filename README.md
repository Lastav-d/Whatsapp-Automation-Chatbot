# 🚛 Whatsapp Automation Chatbot

An AI-powered WhatsApp chatbot built using n8n for logistics booking and transporter onboarding.

---

## 🚀 Features

* 📦 Book Vehicle (Customer Flow)
* 🚛 Provide Vehicle (Transporter Flow)
* 🆘 Support System
* 🔁 Smart session handling (multi-entry per user)
* 📊 Google Sheets integration
* 🤖 AI integration (Gemini - planned)
* 📱 WhatsApp Automation

---

## 🛠 Tech Stack

* n8n (Workflow Automation)
* WhatsApp Cloud API
* Google Sheets
* JavaScript (custom logic)
* HTTP API (for interactive buttons)

---

## 🔄 Workflow Overview

1. User sends message on WhatsApp
2. n8n processes input
3. Stores session in Google Sheets
4. Executes flow (Customer / Provider / Support)
5. Sends response back to WhatsApp
